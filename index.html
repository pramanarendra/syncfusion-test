    Getting Started

    For quick setup of Syncfusion, you can use CDN link and script for all components:

    // Essential JS 2 material theme
<link href="https://cdn.syncfusion.com/ej2/28.1.33/tailwind3.css" rel="stylesheet" type="text/css"</link>
// Essential JS 2 all script
<script src="https://cdn.syncfusion.com/ej2/dist/ej2.min.js" type="text/javascript"></script>
    Notes: Local script and style requires us to make free trial account and download Essential Studio JavaScript
    Style Theme: Syncfusion components available in a few different themes (Material, Bootstrap, Tailwind)

    Grid Component

    The Grid component was one of the first things I learned. Here's my implementation notes:

    Basic Setup

    let grid = new ej.grids.Grid({
    dataSource: data,
    columns: [
        { field: 'id', headerText: 'ID' },
        { field: 'name', headerText: 'Name' },
        { field: 'age', headerText: 'Age' }
    ],
    allowPaging: true,
    pageSettings: { pageSize: 10 }
});
    Note from my experience:Without direct utility class styling, it seems using Tailwind vs vanilla CSS doesn't seems that different
